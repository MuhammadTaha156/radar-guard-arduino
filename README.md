# radar-guard-arduino
An AI-assisted radar detection system using an ultrasonic sensor, servo, LCD, buzzer, and LEDs.

## Hardware
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- SG90 Servo Motor
- I2C 16x2 LCD
- Red & Green LEDs
- Buzzer

## Features
- Scans with servo
- Detects distance
- AI-based frequent movement detection
- Alerts with LEDs and buzzer
- LCD display messages
